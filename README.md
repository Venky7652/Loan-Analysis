# Loan-Analysis
Mini project
# Enhancing Loan Approval Strategies with Predictive Data Analysis

## Project Overview

This project focuses on analyzing loan application data and predicting loan approval status using Machine Learning techniques. The project helps financial institutions make better loan approval decisions by identifying patterns in customer data and assessing loan eligibility.

The project uses:

- Microsoft Excel for data preparation and cleaning
- Python for data analysis and machine learning
- Machine Learning algorithms for loan prediction
- Power BI for dashboard creation and visualization

---

## Project Objectives

- Analyze customer loan data
- Identify factors affecting loan approval
- Build predictive machine learning models
- Improve loan approval decision-making
- Visualize insights using Power BI dashboards

## Project Workflow

### Step 1: Data Collection

Loan applicant data is collected and stored in Excel files.

### Step 2: Data Preprocessing

- Handling missing values
- Removing duplicates
- Feature selection
- Data transformation

### Step 3: Exploratory Data Analysis

- Income Analysis
- Loan Amount Analysis
- Credit History Analysis
- Approval Status Analysis

### Step 4: Machine Learning Model Training

The cleaned data is used to train machine learning models to predict loan approval status.

### Step 5: Model Evaluation

Performance is measured using:

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

### Step 6: Dashboard Creation

Power BI is used to create interactive dashboards for visualization and reporting.
